canon_template_shell

Purpose:
Reusable BlackRoad process template for projects, experiments, campaigns, software builds, launches, research, sales, and personal goals.

Shape:
- 27 macros
- 172 steps
- final repeatability loop

Primary BlackRoad products:
- RoadWork

Implementation rule:
Each macro can become a milestone.
Each step can become a task.
Each task can receive status, owner, due date, awaiting state, action log, and receipt.
